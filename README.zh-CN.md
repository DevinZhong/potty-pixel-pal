# Potty Pixel Pal

[English](README.md) | 简体中文

一个原创的 LCD 如厕训练玩具原型。

Potty Pixel Pal 是一个小型开源网页玩具，主角是 Pippin Roll（昵称 PIP）/ 皮皮卷：一个卷纸造型的电子伙伴，带有薄荷绿 LCD 眼睛、厚实的彩色按钮，以及一个低分辨率的如厕训练反应小游戏。

项目吸收了复古电子学习玩具、LCD 掌机，以及老式卷纸造型电子装置的气质。它与任何现有品牌或权利方都没有关联。仓库中不包含任何第三方美术、截图、Logo、音频或专有素材。

## 项目目标

- 打造一个正面视角的卷纸玩具角色，具有复古 LCD 性格。
- 使用 HTML、CSS 和 JavaScript 构建可在浏览器中游玩的原型。
- 保持素材来源原创且适合开源。
- 清晰记录设计，使项目后续演进不依赖受保护的源素材。

## 当前状态

仓库已经包含一个可游玩的正面视角浏览器原型，代码位于 `index.html`、`src/styles.css` 和 `src/main.js`，同时配有一套精简的双语公开文档。当前视觉目标是一个正面卷纸玩具形象，整体气质偏老式电子玩具。

## 计划体验

- 一个正面朝向的卷纸玩具体块，配亮黄色矩形塑料拉杆。
- 两个圆形 LCD 风格眼睛显示屏。
- 一个矩形薄荷绿 LCD 状态 / 游戏屏。
- 三个有实体感的按钮，对应简单操作。
- 一个基于位置匹配的小游戏。
- 低保真蜂鸣、屏幕闪烁、睡眠模式和玩具式按键反馈。
