# AGENTS.md

## Documentation Policy

- `README.md` is the default landing document and stays in English.
- `README.zh-CN.md` is the Simplified Chinese mirror of the main README.
- The public doc set stays compact.
- `docs/product.md`, `docs/status.md`, and their `docs/zh-CN/` mirrors are the only tracked public docs.
- When updating one public language version, update the other public language version in the same change set whenever the meaning has changed.
- Cross-language links should keep working in both directions.
- Keep internal-only notes, reference comparisons, and IP strategy discussion out of tracked public docs.

## Repo Tracking

- Keep this file tracked in git so future contributors and agents see the bilingual maintenance rule by default.
