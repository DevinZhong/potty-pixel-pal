# Public Status

[简体中文](zh-CN/status.md)

This document separates the current implementation state from the broader product description.

## Implemented

- Front-view paper-roll toy shell in the browser.
- Rectangular yellow pull-handle frame with a 90-degree pull interaction.
- LCD-style eyes, one-line status display, and three toy buttons.
- Basic face switching and short one-line text reactions.
- Hidden reaction mini-game that appears after repeated handle pulls.
- Position-based poop matching for left, center, and right button input.
- Keyboard shortcuts for play and testing.
- Reduced-motion-safe CSS timing fallback.

## In Progress

- Final polish on toy silhouette and material detail.
- Tightening the visual match between the public product description and the live prototype.
- Refining LCD text layout, icon spacing, and decorative screen details.

## Planned

- Web Audio beeps and buzz feedback.
- Stronger LCD ghosting, scan-line, and low-refresh effects.
- Deeper game pacing, combo feedback, and bonus moments.
- More toy-state transitions such as sleep, boot, and richer idle behavior.
- Accessibility review and more explicit control hints where needed.

## Notes

- The product guide describes the intended toy behavior and direction.
- The prototype does not implement every planned effect yet.
- When the implementation and the product guide diverge, this file should clarify the current shipped state.
