# 公开状态

[English](../status.md) | 简体中文

这份文档用来把“当前实现到哪一步”与“产品想要成为什么样”分开说明。

## 已实现

- 浏览器中的正面卷纸玩具外壳。
- 可拉动约 90° 的黄色矩形拉杆框。
- LCD 风格眼睛、单行状态屏、三个玩具按钮。
- 基础表情切换和简短单行文本反馈。
- 通过重复拉动把手进入的隐藏反应小游戏。
- 左、中、右三位置的便便匹配输入。
- 便于游玩和测试的键盘快捷键。
- 面向 reduced-motion 的基础 CSS 回退处理。

## 进行中

- 玩具整体轮廓和材质细节的最终打磨。
- 公开产品文档与当前原型之间的持续对齐。
- LCD 文本排版、图标间距和屏幕装饰细节的继续优化。

## 计划中

- Web Audio 提示音和蜂鸣反馈。
- 更强的 LCD 拖影、扫描线和低刷新效果。
- 更完整的游戏节奏、连击反馈和奖励时刻。
- 更丰富的玩具状态切换，例如 sleep、boot 和更完整的 idle 行为。
- 可访问性复查，以及在需要处补充更明确的控制提示。

## 说明

- `product` 文档负责描述产品方向和目标体验。
- 原型目前还没有实现所有计划中的效果。
- 当实现状态和产品文档存在差异时，以这份文档说明当前公开原型的实际状态。
