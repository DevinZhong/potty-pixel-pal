## 一句话定位

Pippin Roll（昵称 PIP）/ 皮皮卷 是一个卷纸造型的电子如厕训练玩具，通过厚实按钮、缓慢刷新的薄荷绿 LCD 表情，以及一个隐藏的反应小游戏，带玩家完成简单的如厕流程想象。

## 交互模型

Pippin Roll（昵称 PIP）/ 皮皮卷 像一个真实存在的电子如厕训练玩具。玩家不直接点击眼睛或 LCD 屏幕，它们都是输出设备。唯一输入是黄色拉杆和下面三个厚实按钮。

### 控件

| 控件 | 操作方式 | 作用 |
| --- | --- | --- |
| 顶部黄色拉杆 | 向下拖动顶部黄色横杆，直到它从头顶翻到眼睛附近，再松手 | 唤醒、确认、触发表情反应、进入或退出小游戏 |
| `1` 按钮 | 按下 | No. 1 / 小便训练提示 |
| `◎` 按钮 | 按下 | 冲水 / 清理 / 重置提示 |
| `2` 按钮 | 按下 | No. 2 / 大便训练提示 |

只有顶部黄色横杆可以被拖动。左右两条侧杆只是同一拉框的视觉组成部分，不是单独控件。轻点不会触发任何动作。只有当拉杆被向下拖到足够深、接近从顶部翻转 90° 到眼前的位置时，才算一次有效拉动；如果过早松手，它会弹回去并且不生效。

### 主训练循环

```text
拉动把手唤醒玩具。
按 1 进行 No. 1 练习，或按 2 进行 No. 2 练习。
眼睛做出反应，单行 LCD 给出简短提示。
按 ◎ 来冲水、清理，并在 No. 1 或 No. 2 之后给出 `GOOD JOB` 表扬。
重复，直到玩具奖励玩家进入小游戏。
```

### 拉杆行为

| 状态 | 拉动结果 |
| --- | --- |
| 待机 / 训练 | 切换表情并显示简短问候 |
| 犯困 | 被唤醒并眨眼 |
| 连续拉动几次之后 | 进入 `POTTY POP` 小游戏 |
| 小游戏中 | 退出小游戏并回到训练 |

当前原型中的捷径：完整拉动三次可进入小游戏。测试时可按 Space 进入或退出小游戏。

### 输出规则

- 眼睛只是屏幕输出。
- LCD 只是屏幕输出。
- 只有按钮和拉杆是直接输入。
- 玩具优先使用单行短文本，例如 `PIPPIN ROLL`、`NO. 1`、`FLUSH`、`NO. 2`、`GOOD JOB`、`MISS` 和 `POTTY POP`。

## 玩法

### 工作标题

Potty Pop LCD

### 产品幻想

Potty Pixel Pal 首先是一个卷纸造型的电子如厕训练玩具，而不是一个独立街机游戏。反应游戏是这个玩具内部自带的小游戏。

### 主玩具循环

| 按钮 | 训练含义 | LCD 示例反馈 |
| --- | --- | --- |
| 左 / 黄 | No. 1 / 小便 | `NO. 1`, `PEE?`, `TRY 1` |
| 中 / 蓝 | 冲水 / 清理 / 表扬 | `FLUSH`, `CLEAN`, `GOOD JOB` |
| 右 / 红 | No. 2 / 大便 | `NO. 2`, `POOP?`, `TRY 2` |

眼睛会对每次互动做出反应。它们可以眨眼、眯眼、微笑、惊慌、犯困、眨单眼、骄傲等等。它们是输出屏，而不是点击目标。

### 小游戏循环

窄 LCD 状态屏被分成左、中、右三个隐藏位置。一个低分辨率便便图标会随机出现在其中一个位置，玩家必须在倒计时结束前按下正下方对应的实体按钮。

进入小游戏后，按钮不再表示不同的如厕概念，而只是位置输入。

| 屏幕位置 | 按钮 | 含义 |
| --- | --- | --- |
| 左侧出现 | 左 / 黄按钮 | 清掉左侧图标 |
| 中间出现 | 中 / 蓝按钮 | 清掉中间图标 |
| 右侧出现 | 右 / 红按钮 | 清掉右侧图标 |

原型使用原创的块状像素便便图标。

### 控制方式

| 输入 | 动作 |
| --- | --- |
| 键盘 `1` | 左按钮 |
| 键盘 `2` | 中按钮 |
| 键盘 `3` | 右按钮 |
| 触摸 / 点击 | 按下可见按钮 |
| Space | 进入 / 退出小游戏 |

### 状态机

```text
SLEEP
  屏幕变暗、慢速眨眼、低电量滴答
BOOT
  单行扫描、短促开机音、Logo 文案
TRAIN
  No. 1 / No. 2 提示，然后冲水 / 表扬的如厕训练循环
IDLE
  眼睛待机动画，等待输入
GAME
  三位置便便反应小游戏
SCORE
  排名、分数，以及骄傲或不满的反应
```

### 小游戏规则

- 一个便便图标会在三个位置中的某一个出现，并持续很短时间。
- 按对对应按钮会清除图标、加分并累计连击。
- 按错按钮会打断连击并触发错误表情。
- 超时算 miss。
- 每隔几次命中会略微加速，但刷新仍保持老式 LCD 的分步感。
- 后续轮次可以加入诱饵闪烁、双重弹出，或短暂的 `FLUSH` 奖励回合。

### 计分

| 事件 | 分数 |
| --- | --- |
| 正确命中 | +1 |
| 清除 10 个 | 获胜 / 星星庆祝 |
| 错误命中 | 蜂鸣并重置连击 |
| 超时 | Miss 并重置连击 |

### 反馈

- 正确：短促上扬提示音，快乐眼睛。
- 错误：蜂鸣，惊慌眼睛。
- 超时：变暗闪烁。
- 连击：单行 LCD 显示类似 `X3` 的计数。
- 结束：单行文本如 `STAR` 或 `AGAIN?`。

## 语气

整体语气顽皮、小小的、带一点荒诞感。笑点来自一个过分认真地执行超小任务的老旧小装置。
